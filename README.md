# 健身房管理系统 java43

## 项目概述

本项目是基于SpringBoot和Vue的健身房管理系统，采用前后端分离的设计模式，为健身房提供了一站式的管理解决方案。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 系统环境
- 兼容Windows和macOS操作系统

### 推荐运行环境
- jdk1.8
- maven
- MySQL
- nodejs

### 服务端技术
- Springboot
- Mybatis-Plus

### 前端技术
- Vue
- axios
- Element-UI
